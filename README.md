# 🐧 Unix Learning Journey

¡Bienvenido a mi repositorio de aprendizaje! 🚀

En este espacio documento mi progreso, apuntes y resoluciones mientras aprendo a dominar **Unix** y la **línea de comandos (CLI)**. Mi método de estudio principal es la práctica constante a través de distintos *wargames*, juegos y desafíos interactivos.

El objetivo de este repositorio es servir como mi "cerebro externo" y como portafolio de mi evolución técnica.

## 🎮 Desafíos Documentados

Hasta el momento, estoy documentando mi progreso en los siguientes juegos:

* [**Terminus**](./terminus.md): Mis primeros pasos en la terminal y navegación básica.
* [**OverTheWire: Bandit**](./bandit.md): Comandos esenciales de Linux, manejo de archivos, permisos y tuberías.
